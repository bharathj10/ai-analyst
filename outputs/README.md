# Outputs folder

Generated reports, charts, decks, HTML files, and intermediate artefacts go here when you are running a single default project or a quick one-off analysis.

Contents are gitignored except this README and `.gitkeep`.

For multiple separate projects in this repo, save outputs under the project folder instead:

```text
projects/<project-name>/outputs/
```

Example prompt:

```text
Analyse data/member_snapshot.csv and save the report, charts, and findings under outputs/ with the prefix 2026-05-03_salary_sacrifice.
```
